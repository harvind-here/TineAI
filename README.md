# TINE AI

### Terminal based AI agent for Browser automation from NLP instructions.

To set up and run TINE AI, follow these steps in your terminal:

1.  **Activate the Conda environment:**
    ```bash
    conda activate playwright_mcp
    ```

2.  **Run the Playwright command (likely to install/initialize something):**
    ```bash
    npx @playwright/mcp@latest
    ```

3.  **Execute the main agent script:**
    ```bash
    python browser_agent.py
    ```
